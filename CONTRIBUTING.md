# Contributing to Kontainer Companion

Thank you for your interest in contributing! Please follow these guidelines:

- Open an issue to discuss your proposed change before submitting a PR.
- Fork the repository and create your branch from `main`.
- Write clear, concise commit messages.
- Ensure your code passes all tests and analysis (`flutter analyze`).
- Document any new features or changes in the README or relevant docs.
- Follow the code style used in the project.
